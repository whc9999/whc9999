  - `AWS S3 Presigned URL`을 활용하여 대용량 3D 모델 파일 서빙 시 발생하는 백엔드 트래픽 병목 해소.
  - `GitHub Actions`를 이용해 코드 푸시 시 자동으로 빌드, 테스트, EC2 서버 배포가 이루어지는 CI/CD 파이프라인 구축.

  - 프론트엔드-백엔드 도메인 분리 환경에서 발생하는 크로스 도메인 쿠키 정책(CORS) 문제를 해결하여 안정적인 세션 로그인 구현.
  - `Swagger` 도입 및 DTO 최적화(`@JsonAlias`)를 통한 프론트엔드/기획 파트와의 협업 효율성 증대.

  - 유저의 전체 퀴즈 풀이 기록을 바탕으로 총 시도 횟수, 정답 횟수, 정답률을 동적으로 집계하는 맞춤형 마이페이지 API 개발.
